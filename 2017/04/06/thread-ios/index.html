四种方式实现多线程:

pthread

  • 第一个参数:目标对象
  • 第二个参数:选择器,调用哪个方法
  • 第三个参数:前面方法需要传递的参数
    /
    NSThread     thread1 = [[NSThread alloc]initWithTarget:self selector:@selector(run) object:nil];

1) 当run 无参数时, object 可以填null
2) 当run 有参数时, object 可以填null
3) 但run 有参数时(声明run:), selector 必须与run的声明一致 为 run:
